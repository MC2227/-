(1)程式的原理與功能:
程式功能是將文字或是音訊的一般文字轉換成密碼、或將密碼轉換成一般文字，簡單的說就是文字和音訊的摩斯密碼加密及解密，文字的部分就是一個對一個的轉換，音訊部分則是利用聲音為波的性質用numpy產生正弦波讓python去判讀以進行加密。
(3)程式的架構:
def audio_to_text  定義函數是音訊加密的程式
def encrypt、def decrypt、def auto_process 這些定義函數是關於文字的轉換
